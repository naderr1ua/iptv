# IPTV

[![auto-update](https://github.com/naderr1ua/iptv/actions/workflows/auto-update.yml/badge.svg)](https://github.com/naderr1ua/iptv/actions/workflows/auto-update.yml)

Collection of publicly available IPTV (Internet Protocol television) channels from all over the world.

## Usage

![VLC Network Panel](https://github.com/naderr1ua/iptv/raw/master/.readme/preview.png)

To watch IPTV, simply insert one of the links below into any player that supports M3U playlists:

- `https://naderr1ua.github.io/iptv/index.m3u`
- `https://naderr1ua.github.io/iptv/index.nsfw.m3u` (includes adult channels)
- `https://naderr1ua.github.io/iptv/index.category.m3u` (grouped by category)
- `https://naderr1ua.github.io/iptv/index.country.m3u` (grouped by country)
- `https://naderr1ua.github.io/iptv/index.language.m3u` (grouped by language)
- `https://naderr1ua.github.io/iptv/index.region.m3u` (grouped by region)
